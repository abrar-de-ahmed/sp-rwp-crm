## TECH STACK

- Framework: Next.js 14 (App Router) + TypeScript
- Styling: Tailwind CSS + shadcn/ui component library
- Database: PostgreSQL (SQLite in development mode)
- ORM: Prisma
- Authentication: NextAuth.js with JWT sessions
- AI Model: GLM-4 Plus (via z-ai-web-dev-sdk, free tier)
- WhatsApp/Meta: Meta Cloud API (FB Messenger + Instagram DM + WhatsApp)
- Email: Resend (email delivery service)
- Deployment Target: Cloudflare Pages (frontend) + Neon PostgreSQL (production DB)
- Everything is 100% free stack - no paid tools except Meta Cloud API (free tier)
